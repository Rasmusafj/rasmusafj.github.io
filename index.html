Data

In order to generate the data, we had to find different subreddits related to technologies that could be used for the project. During intensive google search and already known knowledge, a list of 49 subreddits was created. The list is obvious very biased and inevitably reflects personal knowledge and interests. However, other more "correct" approaches would require looking at all the reddit users (millions) which seemed infeasible for this particular project.

The final dataset was generated by utilizing two different APIs. First, the pushshift.io was used to mine all submission ids for a period of 1 year for each of the chosen subreddits. Then the Python Reddit API Wrapper (PRAW) was used to mine actual comments with text, author and timestamp properties based on the extracted submission ids. The reason for using two different APIs is that PRAW does not support getting submissions for specific periods.

As previously mentioned, we would like to compare the reddit network reflecting user interests to a network constructed from Wikipedia hopefully reflecting whether technologies are actually related. The Wikipedia network was generated using the Wikipedia API. Edges in the network are generated by searching for links in the actual article texts. Note that the Wikipedia network is saved as a single dictionary, since the actual text is irrelevant for this project.

Network analysis

As previously mentioned, we have networks based on Reddit and a network based on Wikipedia. The Reddit network is created based on user interests. We assume that a user is interested in a subreddit if the user has commented more than 8 times during a time period of 1 year. We are then able to construct a bipartite graph, where user nodes are connected to different subreddit nodes. By doing so, we can project onto the subreddit nodes or we can choose to project onto the user nodes. The projection onto the subreddit nodes is a weighted projection. The weight between two subreddits is determined by the number of users that are connected to both subreddits.

Degree distributions

Since a lot of social networks have shown to be scale-free, i.e. the degree distribution follows a power law, we are interested in deducing whether this is the case for the reddit network, which can be considered a "social network". We will also be looking at the degree distributions for the other networks.


The networks which we have constructed in this project are definitely not scale free. We can see that there are some nodes with high degree in all networks. However, the tendency of having a lot of nodes with a low amount of connections is not really present in any networks. For the users network, a lot of nodes have a large amount of degrees. These nodes/users are of interest since the network might be connected through a small subset of users.

If some of the words sound interesting or familiar, then perhaps the reader should consider looking into that specific technology.

The idea is that by generating word-clouds based on TD-IDF we get word clouds representing frequently used words for each subreddit, that are not also frequently used in all of the other subreddits. Sentiment Analysis

Using the same function as in Assignment 2 to calculate sentiment for comments in each subreddit, we might find something interesting by looking at the means and standard deviations. Initially, we made a hypothesis that the linux community would generally have a more negative sentiment than some of the other subreddits. Is this true? The plot below shows the mean sentiment for each of the subreddits with standard deviation as errorbars.





Clearly, there is close to no difference in the mean of comment sentiments. They are all really close to 5.6 and the standard deviation is also quite similar.

We can also look at the individual distributions to see whether some of subreddits sentiment means are tilted by a large amount of negative comments or positive comments. Perhaps the sentiment of comments is split into multiple groups of subreddit users.




Again, the sentiment of all subreddits is quite similar! Some subreddits contain a slightly larger prorportion of negative comments than some of the others but most of the comments no matter the subreddit have sentiment around 5.6.

Perhaps the reason for this is that the text is often very technical, containing words that are not part of our predefined sentiment list. So the sentiment can be based on the common words that needs to be used in order to explain technical terms and these words probably all have similar sentiment. Therefore, it would be wise to look at other methods for calculating sentiment, since the result that all subreddits contain mean sentiment of around 5.6 is not that useful if you want to find the most positive subreddit to when expanding interests.

Conclusion

In this project, we analyzed networks based on software technologies. We hoped that we were able to find results, that could help technology interested people which new technologies they should learn.

In the network analysis, the most interesting part was the community partitions. Clearly, the detected communities were somewhat related to groups of software technologies e.g. the "math" community or the "front end" community. Both the Wikipedia network and the subreddit network had interesting results. These communities can be used to help guide people towards new technologies to explore. Other results related to betweeness and degree distributions were generally not very interesting.

In the NLP analysis, we generated a word cloud for each of the subreddits. These can be examined and if you find words for a specific subreddit that you think sound interesting or that you've heard of before, then perhaps you should look into that technology. The NLP analysis also included word-clouds over time for a subset of subreddits, sentiment analysis, sentiment analysis over time and word clouds for positive/negative comments, also for a subset.

Generally, there is a lot of bias in the data due to the fact that only a fraction of technology related subreddits were considered. This was due to the fact that both a network and NLP analysis had to be conducted. If more time was given, it would definetly be very interesting to look at all of reddit in a similar approach to see whether we can find communities that are related to different interests.

In the end, the analysis has been quite interesting. We've learned a lot about the different technologies and how they relate. Doing this project, I've myself found some new subreddits to follow, so the project must have been a little successful. The initial goal was to provide a website that easily helps software interested people expand their interests. Even though some results might help with the goal, finding relations between different subreddits was much harder than initially thought.
